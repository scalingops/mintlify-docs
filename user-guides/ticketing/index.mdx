The Tickets module provides comprehensive ticket management integrated with your PSA system. Create tickets using AI assistance, manage templates, and track ticket status all from one interface.

<Note>
The Tickets interface adapts based on your PSA integration (ConnectWise, Halo PSA, Autotask, or Syncro). Some features and field names may vary depending on your connected PSA system.
</Note>

## Accessing Tickets

1. Click **Tickets** in the sidebar
2. The ticket list displays all tickets for the selected company

## Creating a New Ticket

### Starting a New Ticket

<Steps>
  <Step title="Click New Ticket">
    Click the **New Ticket** button in the top right
  </Step>
  <Step title="Use AI Assistant (Optional)">
    Describe the issue in natural language or use voice input
  </Step>
  <Step title="Select Template">
    Choose a template to pre-fill the form
  </Step>
  <Step title="Fill Details">
    Complete the required fields in each section
  </Step>
  <Step title="Submit">
    Click **Create Ticket** to submit
  </Step>
</Steps>

<Frame>
  <img src="/images/ticketing/tickets-new-form.png" alt="New Ticket Form" />
</Frame>

### AI Assistant

The AI Assistant helps you create tickets faster:

| Feature | Description |
|---------|-------------|
| **Natural Language** | Describe the issue in plain English |
| **Voice Input** | Click the microphone to dictate the issue |
| **Auto-Fill** | AI extracts details and pre-fills form fields |
| **Smart Suggestions** | AI suggests appropriate categories and priorities |

<Tip>
Use voice input for hands-free ticket creation. The AI will transcribe your description and extract key details to populate the form.
</Tip>

### Using Templates

Templates provide pre-configured forms for common ticket types:

<Frame>
  <img src="/images/ticketing/tickets-template-selector.png" alt="Template Selector" />
</Frame>

<Steps>
  <Step title="Click Template Dropdown">
    Select the **Template** dropdown in the ticket form
  </Step>
  <Step title="Choose Template">
    Select from available templates like "Generic Issue", "New User Setup", etc.
  </Step>
  <Step title="Complete Fields">
    Fill in the template-specific fields organized in tabs

  <img src="/images/ticketing/tickets-new-template-selected.png" alt="Template Selected" />

Templates organize fields into logical sections displayed as tabs:

- **Technical Triage** - Diagnostic information

  </Step>

## PSA Integration

### Supported PSA Systems

| PSA | Sync Direction | Features |
|-----|----------------|----------|
| **ConnectWise Manage** | Bidirectional | Full ticket sync, time entries |
| **Halo PSA** | Bidirectional | Tickets, actions, SLAs |
| **Autotask** | Bidirectional | Tickets, notes, time |
| **Syncro** | Bidirectional | Tickets, comments |

### Synchronization

Tickets automatically sync with your PSA:
- **New tickets** created in MSPortal appear in your PSA
- **Updates** sync bidirectionally
- **Notes and time entries** are synchronized
- **Status changes** reflect in both systems

## Best Practices

- **Use Templates** - Create templates for common ticket types to ensure consistency
- **Leverage AI** - Use the AI Assistant for faster ticket creation
- **Customize Columns** - Show only the columns you need for cleaner views
- **Regular Review** - Periodically review and update templates
