## Welcome to MSPortal.ai

This quick start guide will help you get oriented with MSPortal.ai and start using the platform effectively in just a few minutes.

## Initial Setup

<Steps>
  <Step title="Sign In">
    Access MSPortal.ai at your organization's URL and sign in with your credentials
  </Step>
  <Step title="Select Company">
    Choose the company you want to work with from the company selector in the top navigation
  </Step>
  <Step title="Navigate the Platform">
    Use the sidebar navigation to access different modules
  </Step>
</Steps>

### Navigation Structure

The sidebar is organized into logical groups:

1. **Overview & Planning**
   - Company Overview - Dashboard view
   - Surveys - Client satisfaction tracking
   - Goals - Strategic objectives
   - Planner - Timeline management
   - Budgets - Financial planning

2. **Engagement**
   - Calendar - Scheduling and events

3. **Service Delivery**
   - Microsoft 365 - M365 management (if integrated)
   - Ticketing - Service requests (if PSA integrated)
   - Compliance - Compliance tracking
   - Training - Training management
   - Devices - Asset management
   - Reporting - Analytics and reports

4. **Administration**
   - Admin - Platform administration (if admin)
   - Settings - Configuration options

## Your First Tasks

### 1. Check the Company Overview

Start by visiting the **Company Overview** to understand:
- Current compliance status
- Recent runs and issues
- Upcoming planner items
- Overall company health metrics

### 2. Review Your Planner

Navigate to **Planner** to see:
- Upcoming tasks and deadlines
- Project timelines
- Project timelines
- Device refresh cycles

<Tip>
Use the timeline view controls to switch between Month, Quarter, and Year views
</Tip>

### 3. Explore Settings

Visit **Settings** to:
- Update your user profile
- Configure preferences
- Set up integrations (if admin)
- Customize your experience

## Common Workflows

### For Managers

1. **Daily Review**
   - Check Company Overview dashboard
   - Review compliance status
   - Monitor upcoming planner items

2. **Weekly Planning**
   - Use Planner to schedule upcoming work
   - Review and create goals from surveys
   - Check report schedules

### For Technicians

1. **Task Management**
   - Check assigned planner items
   - Review compliance checks
   - Update device information

2. **Client Work**
   - Access client-specific settings
   - Review compliance requirements
   - Generate reports as needed

### For Administrators

1. **Platform Management**
   - Configure user accounts
   - Set up integrations
   - Manage company settings

2. **Monitoring**
   - Review system health
   - Check integration status
   - Monitor user activity
