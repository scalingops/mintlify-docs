## How to Use These Guides

### Guide Structure

Each guide follows a consistent structure:

1. **Overview** - What the module does and when to use it
2. **Getting Started** - Essential setup and orientation
3. **Common Tasks** - Step-by-step procedures for daily operations
4. **Advanced Features** - Power user functionality
5. **Troubleshooting** - Solutions to common issues

### Finding Information

<Steps>
  <Step title="Browse by Module">
    Select the guide for the module you're working with from the sidebar or cards above
  </Step>
  <Step title="Use Search">
    Press `Ctrl+K` (or `Cmd+K` on Mac) to search for specific topics
  </Step>
  <Step title="Check Prerequisites">
    Review any prerequisites at the start of each procedure
  </Step>
  <Step title="Follow Along">
    Use the step-by-step instructions with screenshots where available
  </Step>
</Steps>
