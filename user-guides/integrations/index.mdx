The Integrations module enables seamless connectivity between MSPortal.ai and your existing technology stack, allowing data synchronization and workflow automation across platforms.

## Key Features

- **Pre-built Connectors** - Ready-to-use integrations with popular services
- **API Management** - RESTful API for custom integrations
- **Webhook Support** - Real-time event notifications
- **Data Synchronization** - Bi-directional data flow
- **OAuth Authentication** - Secure third-party authentication
- **Integration Monitoring** - Track connection health and data flow

## Getting Started

1. Browse available integrations in the marketplace
2. Select and configure desired integrations
3. Authenticate with third-party services
4. Map data fields between systems
5. Test and activate the integration

## Available Integrations

### Current Integrations

#### PSA & Service Management
- **[ConnectWise Manage](enable-connectwise-integration)** - Complete PSA integration for ticket synchronization, opportunity management, and company/contact sync
- **[Halo PSA](/user-guides/integrations/enable-halo-integration)** - Full-featured integration with Halo PSA for service desk and asset management
- **[Autotask PSA](enable-autotask-integration)** - Professional services automation with ticket sync and resource management
- **[Syncro PSA](enable-syncro-integration)** - Comprehensive PSA and RMM platform integration for MSPs, including ticketing, billing, and remote management capabilities

#### RMM (Remote Monitoring & Management)
- **[Addigy](enable-addigy-integration)** - Apple device management platform for syncing macOS devices, policies, and security status (Beta)
- **ConnectWise Asio** - Cloud-native RMM platform for device monitoring, patch management, and endpoint security
- **Datto RMM** - Remote monitoring and management with automated patch management and ransomware detection
- **[NinjaOne](enable-ninjaone-integration)** - Sync organizations, locations, and devices with device type mapping and auto-import
- **Syncro RMM** - Integrated RMM functionality through Syncro for device monitoring, patch management, and remote access

#### IT Documentation
- **[Hudu](enable-hudu-integration)** - IT documentation platform with articles, assets, passwords (live-fetched), procedures, and Magic Dash compliance push
- **[IT Glue](enable-itglue-integration)** - IT documentation with flexible assets, passwords (live-fetched), and checklists

#### Backup & Disaster Recovery
- **[Acronis Cyber Protect](enable-acronis-integration)** - Unified backup, disaster recovery, and cyber protection with tenant, resource, and alert syncing

#### Security & Compliance
- **[Cork](enable-cork-integration)** - Security and compliance platform syncing compliance events, vulnerabilities, and endpoints

#### Audit & Configuration Management
- **[Liongard](enable-liongard-integration)** - Configuration audit platform that syncs metrics, systems, and detections with compliance metric mapping

#### Productivity & Collaboration
- **Microsoft 365 (Calendar Sync)** - Seamlessly sync calendars, meetings, and scheduling between MSPortal and Microsoft 365

#### AI & Automation
- **OpenAI** - Leverage AI capabilities for intelligent automation, content generation, and enhanced decision-making

### Coming Soon
