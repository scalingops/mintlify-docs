This guide explains how to import users from your PSA integration (ConnectWise Manage, HaloPSA, or Syncro) into MSPortal.ai. You can import users individually or in bulk, and link them to existing MSPortal.ai users.


## Prerequisites

- At least one PSA integration must be configured (ConnectWise Manage, HaloPSA, or Syncro)
- User roles must be created in MSPortal
- You must have "Manage Users" permission

## Steps

### 1. Navigate to Integrations Settings

Go to [https://msportal.ai/settings/integrations](https://msportal.ai/settings/integrations)

![](https://ajeuwbhvhr.cloudimg.io/https://colony-recorder.s3.amazonaws.com/files/2025-09-04/279d628a-e176-4ecd-b8c2-0b2ca6df26d4/ascreenshot.jpeg?tl_px=130,121&br_px=1850,1083&force_format=jpeg&q=100&width=1120.0)


### 2. Access Users and Roles

From the Settings menu, click on **Users and Roles**.

![](https://ajeuwbhvhr.cloudimg.io/https://colony-recorder.s3.amazonaws.com/files/2025-09-04/279d628a-e176-4ecd-b8c2-0b2ca6df26d4/ascreenshot.jpeg?tl_px=0,0&br_px=1719,961&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=292,120)


### 3. Open Import Dialog

Click the **Import** button and select **Import from ConnectWise Manage** from the dropdown menu.

![](https://ajeuwbhvhr.cloudimg.io/https://colony-recorder.s3.amazonaws.com/files/2025-09-04/6f7bdffe-01c2-4b46-997e-6bd15830aac9/ascreenshot.jpeg?tl_px=260,0&br_px=1980,961&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=692,132)


### 4. Review Available Users

The import dialog displays all users from ConnectWise Manage in two columns:
- **Left column:** Available users to import
- **Right column:** Import preview and settings

![](https://ajeuwbhvhr.cloudimg.io/https://colony-recorder.s3.amazonaws.com/files/2025-09-04/41f8620a-2f84-43ac-b5a0-464a7893625d/ascreenshot.jpeg?tl_px=260,0&br_px=1980,961&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=654,165)


### 5. Select Users to Import

Select users to import by:
- Clicking individual users to select them one by one
- Using Ctrl+Click (Windows) or Cmd+Click (Mac) to select multiple users
- Using the search box to filter users by name

![](https://ajeuwbhvhr.cloudimg.io/https://colony-recorder.s3.amazonaws.com/files/2025-09-04/8a0cc949-5715-4de9-a3eb-a275c9d7c3e1/ascreenshot.jpeg?tl_px=0,217&br_px=1719,1178&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=245,276)


### 6. Link to Existing Users (Optional)

For users who already exist in MSPortal (like your primary admin account):

1. Select the ConnectWise user from the left panel
2. In the right panel, click the **Link to Existing User** dropdown
3. Select the matching MSPortal user
4. This will sync the ConnectWise data with the existing account instead of creating a duplicate

![](https://ajeuwbhvhr.cloudimg.io/https://colony-recorder.s3.amazonaws.com/files/2025-09-04/1416d54b-dbc1-429f-a061-8dac5e0e0802/ascreenshot.jpeg?tl_px=0,138&br_px=1719,1099&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=246,277)


### 7. Configure Import Settings

For each selected user, configure:

- **Role:** Select the appropriate MSPortal role
- **Department:** Assign to a department (optional)
- **Send Welcome Email:** Check to send login credentials

![](https://ajeuwbhvhr.cloudimg.io/https://colony-recorder.s3.amazonaws.com/files/2025-09-04/f72fb09a-e6bf-42b3-a095-75ece7bd0d1c/ascreenshot.jpeg?tl_px=260,243&br_px=1980,1205&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=729,285)


### 8. Add Missing Email Addresses

> **Important:** All users must have email addresses in MSPortal. If a ConnectWise user lacks an email address:

1. Click on the user in the import preview
2. Enter their email address in the **Email** field
3. The email will be saved with their MSPortal account

![](https://ajeuwbhvhr.cloudimg.io/https://colony-recorder.s3.amazonaws.com/files/2025-09-04/af58d92d-571c-4150-aff6-ac1cb02512b8/ascreenshot.jpeg?tl_px=0,154&br_px=1719,1115&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=477,277)

### 9. Complete the Import

Once all settings are configured:

1. Review the import summary
2. Click the **Import Users** button
3. Wait for the confirmation message
4. Imported users will appear in your Users list

## Post-Import Tasks

- Verify imported users have the correct roles and permissions
- Send welcome emails if not done during import
- Configure any user-specific settings or preferences
- Set up two-factor authentication for admin users

## Troubleshooting

### Common Issues

- **Missing users:** Ensure the ConnectWise integration has proper API permissions
- **Email required error:** All users must have valid email addresses
- **Role not selected:** Each user must be assigned to a role before import
- **Duplicate user error:** User may already exist - use the link feature instead
