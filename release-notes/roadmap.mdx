## Planned Features

### Expanded Playbook Library
New compliance and operational playbooks are coming soon:
- **Compliance Frameworks**: ISO 27001, CIS Controls, NIST, and more
- **Operational Playbooks**: Client Onboarding, Cyber Insurance Renewals, and other MSP workflow essentials

### New Integrations
We're expanding our integration ecosystem to connect with more tools you use every day:
- **Azure Device Sync** - Sync device information from Azure AD/Entra ID
- **Intune Settings Documentation** - Document and track Intune configuration policies
- **IT Glue Integration** - Connect your IT Glue documentation
- **Hudu Integration** - Sync with Hudu documentation platform
- **Liongard Integration** - Pull configuration data from Liongard
- **NinjaOne Integration** - Connect with NinjaOne RMM
- **Pia Integration** - Integrate with Pia for enhanced workflows
